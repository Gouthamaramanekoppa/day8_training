# day8_training
Trained the neural network for multiple epochs and observed accuracy improvement during learning.

The neural network was trained using the prepared and normalized MNIST dataset. Training is the process where the model learns patterns from data by adjusting its internal weights.

The model was trained for multiple epochs, where one epoch represents one complete pass of the training dataset through the neural network. During each epoch, the model performed forward propagation to generate predictions, calculated the loss by comparing predictions with actual labels, and then used backpropagation to update the weights using the optimizer.

As training progressed across epochs, the accuracy gradually improved while the loss decreased. This indicated that the model was successfully learning meaningful patterns from the dataset. Observing accuracy improvement helped in understanding how neural networks refine their predictions over time.

Additionally, training metrics such as training accuracy and loss were monitored after every epoch. This provided insight into whether the model was learning effectively or facing issues such as slow convergence.

By the end of Day 8, a clear understanding was developed about how training works in deep learning, how epochs influence performance, and how models improve progressively through repeated exposure to data.
